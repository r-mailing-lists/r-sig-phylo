# r-sig-phylo

Phylogenetic methods

Automated mbox archive. Updated daily via GitHub Actions.

## Structure

- `raw/` - Monthly .mbox files downloaded from [https://stat.ethz.ch/pipermail/r-sig-phylo/](https://stat.ethz.ch/pipermail/r-sig-phylo/)
- `processed/` - Structured JSON output from rmail-parser
- `meta.json` - List statistics

## Viewer

Browse these archives at [r-mailing-lists.github.io/viewer](https://r-mailing-lists.github.io/viewer/lists/r-sig-phylo/)
